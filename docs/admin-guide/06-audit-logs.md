# Section 6: Audit Logs

The **Audit Logs** section provides a timestamped record of all activity and system events carried out within your institution's Payforms account. To access it, click **Audit Logs** in the left navigation sidebar.

![Figure 47: The Audit Logs page](/payforms-docs/assets/images/Picture47.png)

Each entry in the log table contains the following columns:

- **Timestamp** — The exact date and time the action was performed
- **Action** — A label describing what was done (e.g. View Contacts, View Transactions, View Reports Summary)
- **Entity** — The area of the platform the action was performed on (e.g. Contact, Transaction, Report)
- **Actor** — The email address and role of the user who performed the action
- **IP Address** — The IP address from which the action was carried out

---

## Searching and Filtering Logs

You can narrow down the log entries using the following filters at the top of the page:

- **Search** — Search log entries by keyword
- **Action** — Filter by the type of action performed
- **Entity Type** — Filter by the area of the platform affected
- **Contact ID** — Filter logs associated with a specific contact
- **Date Range** — Set a start and end date to view logs within a specific period

---

## Exporting Logs

To download the audit log, click **Export Logs** in the top-right corner of the page. The log will be downloaded as a CSV file.
